⚙️ Configuración inicial

Para que todos los participantes compartan el mismo leaderboard, ingresa tus credenciales de Supabase. Es gratis — crea tu proyecto en supabase.com, copia el URL y la clave anon public.

También puedes usar el modo local (solo en este navegador) si no tienes Supabase todavía.

🔒 Tus picks se bloquean al inicio de cada partido. Los picks de otros participantes solo son visibles después de que el partido inicia.
Los resultados se actualizan automáticamente vía API cada 5 minutos durante el torneo. También puedes ingresar manualmente si lo necesitas.

Cómo leer las líneas de apuesta

Cada partido muestra las líneas moneyline americanas de casas de apuesta reales (DraftKings / BetMGM).

México -215  El número negativo indica el favorito. Debes apostar $215 para ganar $100.

Empate +340  El número positivo indica el underdog. Apuestas $100 y ganas $340.

Sudáfrica +650  Entre mayor el número positivo, mayor underdog.

O/U 2.5  Over/Under — el total de goles esperados en el partido.
